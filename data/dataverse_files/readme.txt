This dataverse contains data sets in Stata 11 format used as examples in the book: 

Hayes RJ, Moulton LH. Cluster Randomised Trials, 2nd Edition.
 Chapman & Hall/CRC 2017; Boca Raton, FL.

Also included are .hlp files that describe the contents of the data files. These
may be copied in the appropriate subfolders of a Stata installation for ready 
reference. For example, share.hlp could go in c:\ado\personal\ in a Windows 
installation.

The data set names, in alphabetical order, are:

ghana_bednet
laviiswa
mkvtrial
mwanza_stdtrial
mwanza_stdtrial_community
pneumovac
share
simpairs
thrio
zamstar
